Description
In this Project, we build an app to encrypt and decrypt files. We use symmetric ciphers such as AES, DES and RC4 methods in the Python language. We are using Cipher Block Chaining(CBC) and Counter(CTR) operation for comparing DES and AES. For AES, we use library and manual for testing running time library and manual. All of these methods are compiled to Server and Client for data transmission.
